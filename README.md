# AI-Enabled-Personal-Finance-Advisor

A full-stack personal finance management applicatiob that combines financial tracking, analytics dashboards , ML predictions , and a Gemini-powered AI assistant to help users monitor and improve financial health.

FEATURES :
   User Authentication
   Income Management
   Expense Tracking
   Budget Management
   Financial Dashboard
   Analytics & charts
   ML-based Expense Prediction
   ML-based Savings Prediction
   Gemini AI Financial chatbot
   Budget Utilization Tracking
   Monthly Insights
